подпоследовательность у нас такая где а-житый меньше чем а-итый
![[Pasted image 20220904210809.png]]
``` python 
def lis(a):
    f = [0] * (len(a)+1)
    for i in range(1, len(a)+1):
        m = 0 
        for j in range(0, i):
            if a[i-1] > a[j-1] and f[j] > m:
                m = f[j]
        f[i] = m + 1
    return f[-1]


```
