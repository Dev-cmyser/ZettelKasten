## проверка отсортированности за O(len(a))
``` python 


def check_sorted(a, asscending=True):
    flag = True
    n = len(a)
    s = 2 * int(asscending) - 1
    for i in range(0, n-1):
        if s*a[i] > s*a[i+1]:
            flag = False
            break
    return flag

```
