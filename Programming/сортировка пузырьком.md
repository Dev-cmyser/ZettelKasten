4 > 1 -> пеменйтесь -> [2,1,4,3,5]
		4 >3 -> поменятесь -> [2,1,3,4,5]
``` python 

def bubble_sort(A):
	N = len(A)
	for bypass in range(1, N):
		for k in range(0, N-bypass):
			if A[k] > A[k+1]:
				A[k], A[k+1] = A[k+1], A[k]

```
